# sql_customer_behavior_analysis

## Objectif du projet 
Analyser le comportement des clients pour identifier des opportunités business.

## Questions principales
  
- Qui sont les meilleurs clients ?  (Comment les membres annuels et les utilisateurs occasionnels utilisent-ils les vélos Cyclistic différemment ?)
  
- Quels clients risquent de partir ?  (Pourquoi les utilisateurs occasionnels achèteraient-ils des abonnements annuels Cyclistic ?)
  
- Quels produits/services performent le mieux ?   (Comment Cyclistic peut-il utiliser les médias numériques pour inciter les utilisateurs occasionnels à devenir membres ?)

## Outils
- SQL
- Dataset public Cyclistic bike
